# BB2B (bili-bi2bi)
> bilibili赛事竞猜辅助脚本

使竞猜更易于判断和修复了部分bilibili增强脚本的夜间模式在此页面的兼容问题

点击[这里](https://github.com/D4box/BB2B/raw/main/bb2b.user.js)安装

功能列表(已做/想做)
* [X] 明显显示双方赔率
* [X] 夜间模式兼容
* [X] 默认投10个币
* [ ] 更好的投币数量控制
* [ ] 一键自动全投

## 效果测试

2023/6/20 ~ 2023/6/30 共投入400颗硬币，收回388.4枚硬币（倒贴）~~竞猜有风险，投币须谨慎~~
